# 🚨 Execution Rules (Project Level)

## Scope Control

- NEVER create new project unless explicitly asked
- ONLY modify listed files
- NEVER delete existing components

---

## Execution Control (新增)

Before any code change:

1. MUST list:
   - Files to modify
   - Exact changes

2. WAIT for confirmation

3. Then apply changes

- If unsure → ASK
- DO NOT refactor unrelated code
- Preserve imports
